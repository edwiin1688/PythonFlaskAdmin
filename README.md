# 🐍 PythonFlaskAdmin

> 🚀 一個基於 Python Flask 的後台管理系統基礎架構。

## 📋 專案必備檔案清單

為了確保專案能穩定運行與順利部署，請確認包含以下檔案：

- **`requirements.txt`** 📦：列出所有 Python 套件依賴，這是 Docker 建置時的關鍵。
- **`app.py`** 💡：專案的主程式入口。
- **`views.py`** 🛣️：存放 Blueprint 或路由相關邏輯。
- **`__init__.py`** 📂：若 `src` 資料夾是 Python package，建議加上此檔案。
- **`README.md`** 📖：專案說明文件，方便他人了解環境與使用方式。
- **`.dockerignore`** 🚫：列出不需要加入 Docker image 的檔案（例如 `.git`、`__pycache__` 等）。
- **`config.py`** ⚙️：(選用) 存放全域設定參數。
- **`static/` & `templates/`** 🎨：分別存放靜態檔案和 HTML 模板（如有前端需求）。

## 🛠️ 開發與安裝

### 📝 產生與安裝依賴

使用傳統方式產生 `requirements.txt`：
```bash
pip freeze > requirements.txt
```
*這會把目前環境所有已安裝套件及版本寫入檔案。*

安裝依賴：
```bash
pip install -r requirements.txt
```

### ✨ 使用 `uv` 高效開發

1. **建立 uv 虛擬環境並安裝依賴**：
   ```bash
   uv venv
   uv pip install -r requirements.txt
   ```

2. **啟動 Flask**：
   請依照您的環境設定環境變數（範例為 Windows PowerShell）：
   ```powershell
   $env:PYTHONPATH="E:\GitHub\chiisen\PythonFlaskAdmin\src"
   $env:FLASK_APP="src/app.py"
   .venv\Scripts\activate
   python -m flask run
   ```
